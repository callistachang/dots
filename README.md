# dotfiles

Managed with [chezmoi](https://www.chezmoi.io/), install with [these instructions](https://www.chezmoi.io/quick-start/#using-chezmoi-across-multiple-machines) ✨

## Stack

### Coding

- VSCode (general purpose IDE)
- Visual Studio (C++ IDE)
- Kitty (terminal)
- JetBrains Mono (font)
- Windows Subsystem for Linux <- Ubuntu 20.04 (terminal environment)

### Productivity

- Obsidian (notepad)
- Notion (another notepad)
- Anki (flashcards)

### Essentials

- Firefox (browser)
- Spotify (music)
- AutoHotKey (key remapping)
- Zoom (video calling)
- Discord (video calling)
- Telegram Desktop (instant messaging)
- PowerToys (window manager)
- Microsoft Office Suite (yep)
- qBittorrent (bittorrent client)
- Adobe Acrobat DC (PDF editor)
